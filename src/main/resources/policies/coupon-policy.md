# 쿠폰 정책

## 주문 취소 시 쿠폰 복구

주문에 사용된 쿠폰은 해당 주문이 정상적으로 취소되면 자동으로 복구되어 다시 사용할 수 있습니다.
쿠폰 복구는 취소 처리 완료 시점에 즉시 이루어지며, 별도로 신청하실 필요가 없습니다.
복구된 쿠폰의 유효기간은 원래 쿠폰의 유효기간을 그대로 따릅니다.

## 쿠폰 복구가 불가능한 경우

다음의 경우에는 쿠폰이 복구되지 않습니다.

- 쿠폰의 유효기간이 이미 만료된 경우
- 일회성 프로모션 쿠폰으로 복구 불가로 명시된 경우
- 부분 취소로 인해 남은 주문 금액이 쿠폰의 최소 사용 금액에 미달하는 경우

## 반품 시 쿠폰

배송 완료 후 반품하는 경우에도 쿠폰은 동일하게 복구됩니다.
다만 반품 사유가 단순 변심인 경우 왕복 배송비가 차감된 후 환불되며, 쿠폰 복구 자체에는 영향이 없습니다.

## 쿠폰 중복 사용

하나의 주문에는 할인 쿠폰 1장과 배송비 쿠폰 1장을 함께 사용할 수 있습니다.
동일 종류의 쿠폰은 중복 사용이 불가능합니다.
